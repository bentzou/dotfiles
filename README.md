### TL;DR
Run `theresnoplacelikehome` to auto-configure the shell:
- bash
  - autocompletions
  - input handling
  - helper functions
  - ps1 (smart prompt)
  - rc
- git
- ssh
- tmux
- vim


### Quickstart
1. Clone the repo:

   ```git clone git@github.com:bentzou/dotfiles.git```

3. Source the script containing the init function:

   `. setuphome`
   
5. Initialize:

   `theresnoplacelikehome`
